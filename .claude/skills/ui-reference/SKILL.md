# Component Library

- Shared prop groups are defined in [standard props](./reference/standard-props.md).
- Source code at `packages/lib/ui/src/components/{name}/{name}.tsx`.

## Layout

- [Container](./reference/container.md): Cap content width and align it horizontally.
- [Flex](./reference/flex.md): Flexbox container.
- [Grid](./reference/grid.md): CSS Grid container.
- [Inset](./reference/inset.md): Bleed content past a Card's padding.
- [Section](./reference/section.md): Landmark `<section>` with vertical padding.

## Typography

- [Blockquote](./reference/blockquote.md): Block-level quotation set off by a colored leading rail.
- [Code](./reference/code.md): Inline monospace snippet.
- [Em](./reference/em.md): Stress-emphasized inline text.
- [Heading](./reference/heading.md): Semantic headings.
- [Link](./reference/link.md): Inline navigation link for in-app routing.
- [Quote](./reference/quote.md): Short inline quotation.
- [Strong](./reference/strong.md): Strongly-emphasized inline text.
- [Text](./reference/text.md): Body copy, labels, and inline text.

## Components

- [Avatar](./reference/avatar.md): User image with initials/icon fallback.
- [Badge](./reference/badge.md): Compact label or status indicator.
- [Button](./reference/button.md): Styled button.
- [Callout](./reference/callout.md): Informational message.
- [Card](./reference/card.md): Surface container.
- [Checkbox](./reference/checkbox.md): Tri-state checkbox with optional inline label.
- [CheckboxCards](./reference/checkbox-cards.md): Group of checkbox options rendered as selectable cards.
- [DataList](./reference/data-list.md): Description list for label–value pairs.
- [IconButton](./reference/icon-button.md): Square button for a single icon.
- [Kbd](./reference/kbd.md): Inline keyboard input.
- [LinkButton](./reference/link-button.md): Anchor styled as a button for navigation actions.
- [Progress](./reference/progress.md): Linear progress bar.
- [RadioCards](./reference/radio-cards.md): Group of radio options rendered as selectable cards.
- [RadioGroup](./reference/radio-group.md): Group of radios sharing a single value.
- [ScrollArea](./reference/scroll-area.md): Styled overflow container with custom scrollbars.
- [Separator](./reference/separator.md): Visual divider between sibling content.
- [Slider](./reference/slider.md): Range input with one or more thumbs.
- [Switch](./reference/switch.md): Two-state toggle button.
- [TabNav](./reference/tab-nav.md): Horizontal nav strip of routing links.
- [Table](./reference/table.md): Tabular data with optional surface chrome.
- [Tabs](./reference/tabs.md): Switch between panels under a row of triggers.
- [TextArea](./reference/text-area.md): Multi-line text input.
- [TextField](./reference/text-field.md): Single-line text input with optional inline slots.
- [Tooltip](./reference/tooltip.md): Floating label describing its trigger.
