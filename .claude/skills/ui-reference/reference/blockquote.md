# Blockquote

Block-level quotation set off by a colored leading rail.

## Props

Base: `<blockquote>` attributes, margin props, truncate props, wrap props, selectable props, skeleton props.

- `size`: Visual size. `1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9`. Inherits from the parent when omitted.
- `weight`: Font weight. `'light' | 'regular' | 'medium' | 'bold'`.
- `color` (=`'accent'`): Semantic color for the leading rail. `'accent' | 'neutral' | 'danger' | 'warning' | 'success'`.
