# Section

Tag-locked to `<section>`. Applies vertical padding to set page rhythm. Pair with `Container` to cap horizontal width.

## Props

Base: native `<section>` attributes, margin props, skeleton props.

- `size` (=`3`): Vertical padding preset. `1 | 2 | 3 | 4`. Maps to 24px / 40px / 64px / 80px (top + bottom).
