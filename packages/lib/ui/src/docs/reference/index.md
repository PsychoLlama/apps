# Component Library

- Shared prop groups are defined in [standard props](./standard-props.md).
- Source code at `src/ui/components/{name}/{name}.tsx`.

## Layout

- [Container](./container.md): Cap content width and align it horizontally.
- [Flex](./flex.md): Flexbox container.
- [Grid](./grid.md): CSS Grid container.
- [Inset](./inset.md): Bleed content past a Card's padding.
- [Section](./section.md): Landmark `<section>` with vertical padding.

## Typography

- [Em](./em.md): Stress-emphasized inline text.
- [Heading](./heading.md): Semantic headings.
- [Strong](./strong.md): Strongly-emphasized inline text.
- [Text](./text.md): Body copy, labels, and inline text.

## Components

- [Badge](./badge.md): Compact label or status indicator.
- [Button](./button.md): Styled button.
- [Callout](./callout.md): Informational message.
- [Card](./card.md): Surface container.
- [DataList](./data-list.md): Description list for label–value pairs.
- [IconButton](./icon-button.md): Square button for a single icon.
- [Kbd](./kbd.md): Inline keyboard input.
- [Separator](./separator.md): Visual divider between sibling content.
- [Switch](./switch.md): Two-state toggle button.
- [TabNav](./tab-nav.md): Horizontal nav strip of routing links.
- [Tabs](./tabs.md): Switch between panels under a row of triggers.
- [TextArea](./text-area.md): Multi-line text input.
- [TextField](./text-field.md): Single-line text input with optional inline slots.
