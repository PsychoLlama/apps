# Badge

Compact label or status indicator. Inline; pill-shaped by default.

## Props

Base: `<span>` attributes, margin props.

- `size` (=`1`): Visual size. `1 | 2 | 3`.
- `variant` (=`'soft'`): Visual treatment. `'solid' | 'soft' | 'surface' | 'outline'`.
- `color` (=`'accent'`): Semantic color. `'accent' | 'neutral' | 'danger' | 'warning' | 'success'`.
- `radius` (=`'full'`): Corner rounding. `'none' | 'small' | 'medium' | 'large' | 'full'`.
- `highContrast` (=`false`): High-contrast text for stronger emphasis.
