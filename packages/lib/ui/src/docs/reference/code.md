# Code

Inline monospace snippet — names, identifiers, short literals.

## Props

Base: `<code>` attributes, margin props, truncate props, wrap props, selectable props, skeleton props.

- `size`: Visual size. `1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | 9`. Inherits font-size from the parent when omitted.
- `variant` (=`'soft'`): Visual treatment. `'solid' | 'soft' | 'outline' | 'ghost'`.
- `color` (=`'accent'`): Semantic color. `'accent' | 'neutral' | 'danger' | 'warning' | 'success'`.
- `weight`: Font weight. `'light' | 'regular' | 'medium' | 'bold'`.
