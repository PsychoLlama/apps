# Button

Extends `<button>` attributes and [MarginProps](./standard-props.md).

## Props

- `size` (=`2`): Visual size. `1 | 2 | 3 | 4`.
- `variant` (=`'solid'`): Visual treatment. `'solid' | 'soft' | 'outline' | 'ghost'`.
- `color` (=`'accent'`): Semantic color. `'accent' | 'neutral' | 'danger'`.
