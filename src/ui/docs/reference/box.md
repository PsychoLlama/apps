# Box

Low-level surface primitive. Prefer [Flex](./flex.md) or [Grid](./grid.md).

## Props

Base: rendered element attributes, margin props, padding props.

- `as`: HTML box tag. `'div' | 'section' | 'article' | 'aside' | 'nav' | 'main' | 'header' | 'footer' | ...`
- `background`: Surface color. `'page' | 'panelSolid' | 'panelTranslucent' | 'surface'`.
- `radius`: Border radius. `1 | 2 | 3 | 4 | 5 | 6 | 'full'`.
- `shadow`: Box shadow elevation. `1 | 2 | 3 | 4 | 5 | 6`.
