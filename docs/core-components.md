# Core Components

Priority list for the initial component library. Radix UI is the reference implementation.

## Components

1. **Button** — Primary interactive element. Solid, outline, and danger variants.
2. **Callout** — Non-dismissible message banner with icon, title, and description. Warning/error/info intents.
3. **Text** — Typography primitive that consumes the type scale. Eliminates manual fontSize/lineHeight/letterSpacing triplets.
4. **Heading** — Semantic heading built on Text with heading defaults.

## Icons

1. **Warning** — Alert/exclamation indicator for callouts.
2. **Play** — Playback triangle for recording thumbnails.
3. **Close** — × mark for dismiss and remove actions.

## Colors

These palettes are needed beyond the existing accent (blue) and neutral (slate):

1. **Red** — Danger/destructive actions, recording indicator.
2. **Amber** — Warning callout icon.
3. **Green** — Live/active status indicators.
